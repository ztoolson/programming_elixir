Elixir Basics
-------------

Goals of the chapter:

- Five Value types
  - Arbitrary-sized integers
  - Floating-point numbers
  - Atoms (ruby symbols)
  - Ranges
  - Regular Expressions (technically not a type)
- Two System types
  - PIDs and ports
  - References
- Four collection types
  - Tupes
  - Lists
  - Maps
  - Binaries
- Naming, operators, etc.
